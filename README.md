# Retrofit-Example
##### Retrofit2을 이용한 날씨 안드로이드앱 실습 🌈☀️☃️☔️

- 소개

  **Retrofit을 이용하여 안드로이드 날씨 어플리케이션**

  - 실시간으로 날씨 API를 받아와 화면에 출력.
  - 해당 날씨마다 배경색 아이콘 등 다양하게 바뀌게 함.

  

- 사용기술

  ```
  ✔️ 언어: Kotlin
  ✔️ 기기에서 현재 위치 받아옴. 
  ✔️ retrofit2을 통해 날씨 API 요청.
  ```

  

- 실습 방법

  > - retrofit_sample 브런치 clone
  > - https://openapi.sk.com 에서 Weather Planet 구매(무료) - > key 발급
  > - 가이드 대로 코드 작성
